# itorbit-demo

Демонстрационные базы данных для проекта itOrbit::DataComparer (https://www.itorbit.ru/datacomparer/).

Используемые СУБД: PostgreSQL и MySQL.
В каждой базе по две таблицы table1 и table2, данные в которых немного различаются для наглядного представления в приложении DataComparer.

Запускайте 
```
sudo docker compose up --build
```
и подключайтесь к базам через приложение DataComparer.
